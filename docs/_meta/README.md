# Vault internals

Stuff Obsidian uses but humans rarely touch.

- `.obsidian/` — Obsidian config (shared theme + plugins). Workspace state is per-user and gitignored.
- `_templates/` — Templater templates. Spawn ADRs / feature pages from here.
- `_assets/` — Images embedded inline in vault pages (logos, icons, screenshots).

Entry point is [[../Home]].
